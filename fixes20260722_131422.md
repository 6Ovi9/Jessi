## 1. Capa de Firmware C++ (`nexus_halo/`)

### 1.1 `nexus_halo/nexus_halo.ino` (Líneas 1155–1158)
* **Problema:** En el modo RADAR, al girar la muñeca a la derecha (sentido horario), el LED indicador giraba hacia la derecha debido a una resta invertida del rumbo de la brújula.
* **Línea Original (L1155-1157):**
```cpp
    float relative_bearing =
        current_bearing - (compass.isConnected() ? compass.getHeading() : 0.0f);
    relative_bearing = fmod(relative_bearing + 360.0f, 360.0f);
```
* **Línea Corregida:**
```cpp
    float relative_bearing =
        current_bearing + (compass.isConnected() ? compass.getHeading() : 0.0f);
    relative_bearing = fmod(relative_bearing + 360.0f, 360.0f);
```
* **Explicación:** Sumar el `heading` de la brújula compensa la rotación física del reloj en sentido inverso. Al girar a la derecha (+45°), la aguja de indicación gira a la izquierda (-45°) manteniendo la posición relativa hacia la pareja.

## 2. Capa Móvil Flutter & Engine Nativo (`app/lib/`)

### 2.1 `app/lib/services/sync_service.dart` (Detección de Tailscale & Modo Offline)
* **Nuevo método `checkTailscaleConnection()`:**
```dart
  bool _isTailscaleConnected = false;
  bool get isTailscaleConnected => _isTailscaleConnected;

  Future<bool> checkTailscaleConnection() async {
    try {
      final interfaces = await NetworkInterface.list();
      for (final interface in interfaces) {
        if (interface.name.toLowerCase().contains('tun') ||
            interface.name.toLowerCase().contains('tailscale')) {
          _isTailscaleConnected = true;
          notifyListeners();
          return true;
        }
        for (final addr in interface.addresses) {
          if (addr.address.startsWith('100.')) {
            _isTailscaleConnected = true;
            notifyListeners();
            return true;
          }
        }
      }
    } catch (e) {
      debugPrint('[SYNC] Error checking Tailscale interfaces: $e');
    }
    _isTailscaleConnected = false;
    notifyListeners();
    return false;
  }
```
* **Explicación:** Inspecciona las interfaces de red del dispositivo Android en busca de direcciones IP en el rango CGNAT de Tailscale (`100.x.y.z`) o interfaces virtuales `tun`/`tailscale`. Si no se detectan, conmuta a modo Offline.

### 2.3 `app/lib/services/location_service.dart` (Offline-First Local Calculation)
* **Modo Offline-First:**
```dart
  if (!syncService.isTailscaleConnected) {
    // Modo Offline: Usar la última ubicación guardada de la pareja de SharedPreferences
    final cachedPartner = prefs.getString('cached_partner_location');
    if (cachedPartner != null) {
      final partnerLoc = LocationItem.fromJson(jsonDecode(cachedPartner));
      _bearingDeg = BearingCalculator.calculateBearing(myPos, partnerLoc);
      _distanceKm = BearingCalculator.calculateDistance(myPos, partnerLoc);
      onLocationUpdate?.call(myPos, _bearingDeg, (_distanceKm * 1000).round());
    }
  }
```
* **Explicación:** Si Tailscale está desconectado, el teléfono no detiene el servicio ni arroja excepciones. Calcula localmente la distancia y rumbo respecto a la última posición conocida y los transmite por BLE al reloj.

```dart
  onPressed: () async {
    ScaffoldMessenger.of(context).showSnackBar(
      const SnackBar(content: Text('Sincronizando estado completo...')),
    );
    final sync = context.read<SyncService>();
    final loc = context.read<LocationService>();
    final ble = context.read<BleService>();
    
    await sync.checkTailscaleConnection();
    await loc.forceLocationUpdate();
    final partnerLoc = await sync.fetchPartnerLocation();
    if (partnerLoc != null) loc.updatePartnerLocation(partnerLoc);
    await sync.syncConfigWithTimestamps();
    if (loc.currentPos != null && loc.partnerPos != null) {
      ble.writeBearing(loc.bearingDeg);
      ble.writeDistance((loc.distanceKm * 1000).round());
    }
  }
```
